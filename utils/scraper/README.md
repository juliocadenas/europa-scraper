# Scraper Module Structure

This directory contains the refactored scraper modules, organized by responsibility:

- `controller.py` - Main controller coordinating the scraping process
- `browser_manager.py` - Handles browser initialization and management
- `search_engine.py` - Handles searching on cordis.europa.eu
- `content_extractor.py` - Extracts content from web pages
- `text_processor.py` - Processes and analyzes text content
- `result_manager.py` - Manages scraping results and storage
- `progress_reporter.py` - Handles progress reporting
- `url_utils.py` - URL handling and validation utilities
